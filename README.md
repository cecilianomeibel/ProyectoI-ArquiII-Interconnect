# ProyectoI-ArquiII-Interconnect
Repositorio en el cual se trabajara el Proyecto I correspondiente al curso de Arquitectura de Computadores II del Instituto Tecnologico de Costa Rica

# Requisitos del sistema
- Sistema operativo Linux.
- Versión de gcc 9 o superior.
- GNU Make.

# Pasos para ejecutar el proyecto
- En una terminal abierta en la carpeta del proyecto se debe ejecutar el comando make.
- Esto generá el archivo ejecutable del proyecto.
- Para ejecutar dicho archivo deberá ingresar el comando ./main en terminal.
